# Bank Marketing Analysis（销售数据分析和流程优化）

This repository contains the analysis of the Bank Marketing dataset from UCI Machine Learning Repository. 

## Contents
- `data/` : contains `bank-full.csv`
- `bank market dataset.rmd` : R Markdown analysis file
- `bank market dataset.pdf` : PDF report generated from R Markdown

## Overview
The project uses a Random Forest model to predict whether clients will subscribe to a term deposit. Key steps include:
- Data preprocessing
- Model training and evaluation
- Threshold optimization using ROC and Youden Index
- Feature importance analysis
- Business insights and recommendations
